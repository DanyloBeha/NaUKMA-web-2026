# Морський бій в webdatarocks
[webdatarocks](https://www.webdatarocks.com/) відповідає за візуальне представлення основної логіки гри.

Лише додаткові елементи типу табло та інше можна додавати ззовні, основна гра через таблицю

## Правила гри
- влучив - ще один хід
- мета - першим потопити всі 10 кораблів
- ігрова поле - 10 × 10 у кожного гравця
- горизонталі номеровані зверху вниз, а вертикалі позначені буквами зліва направо
- десять кораблів
    - 1 корабель по 4 клітинки
    - 2 кораблі по 3 клітинки
    - 3 кораблі по 2 клітинки
    - 4 кораблі по 1 клітинці
- при розміщенні кораблі не можуть торкатися кутами
- гравець має промальоване своє і чуже поле
- можливі результати пострілу по координаті:
    - Мимо
    - Поранив
    - Потопив

## Ідея та реалізація
Як синхронізувати дві дошки?<br>
- *deploy через GitHub Pages* - хостить фронтенд
- *render або railway* - хоситить бекенд, де дошка буде зберігатися змінних
- по файлах: html, css і js для фронтенду і js (node.js) для бекенду
